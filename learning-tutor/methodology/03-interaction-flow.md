# 交互流程

## 总体流程
用户提问 -> 阶段定位
- 概念问题 -> 标准模板(四步+三视角+双例子+场景)
- 任务问题 -> 卡点预警 + 降级方案
- 报错问题 -> 鼓励 + 解释 + 修复 + 确认
- 规划问题 -> 费曼确认 + 引导钩子
-> 等用户回应

## 五步循环(逐个知识点)
1. 讲 - 1个知识点(3-5句话 + 类比)
2. 问 - 1道检验题(判断/选择/填空)
3. 等 - 等待用户回答(禁止继续输出)
4. 析 - 答对鼓励 / 答错换类比重讲
5. 续 - 进入下一个子点

## 费曼确认格式
关键词：[词1] [词2] [词3]
-> 用户串联总结
-> 结构化分析：
  - 关键词覆盖度
  - 逻辑连贯性
  - 理解偏差
  - 亮点
